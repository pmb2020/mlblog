•类文件采用驼峰法命名（首字母大写），其它文件采用小写+下划线命名
•类名和类文件名保持一致，统一采用驼峰法命名

•类的命名采用驼峰法（首字母大写），
•函数的命名使用小写字母和下划线（小写字母开头）的方式
•方法的命名使用驼峰法（首字母小写），
•属性的命名使用驼峰法（首字母小写），

•数据表和字段采用小写加下划线方式命名

1、文章表

status      状态：0默认 1置顶 2加密

comment_num评论数

CREATE TABLE `wang_mlblog`.`ml_article` ( `id` INT NOT NULL AUTO_INCREMENT , `status` TINYINT(4) NOT NULL DEFAULT '0' , `type` TINYINT(4) NOT NULL , `title` VARCHAR(100) NOT NULL , `content` TEXT NOT NULL , `time` TIMESTAMP NOT NULL , `read_num` INT NOT NULL DEFAULT '0' , `comment_num` INT NOT NULL DEFAULT '0' , `url` VARCHAR(100) NOT NULL , PRIMARY KEY (`id`)) ENGINE = InnoDB;
